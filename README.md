## How It Works

1. Connect your data sources in Two Minute Reports.
2. Add the TMR MCP Server to your AI client (Claude, ChatGPT, etc.).
3. Ask a question about your data in plain English.
4. Get answers with summaries, tables, and charts.

That's it. No code, no queries, no config files.

## How It's Different

|                   | Two Minute Reports MCP               | Typical Ad Platform MCPs                           |
| ----------------- | ------------------------------------ | -------------------------------------------------- |
| **Setup**         | 2 minutes, no code                   | Clone repos, install dependencies, manage API keys |
| **Data sources**  | 22 platforms in one server           | One platform per server                            |
